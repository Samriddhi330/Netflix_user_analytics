# Netflix User Analytics

## Overview

This project analyzes user demographics, viewing behavior, subscription details, and spending patterns to gain business insights and predict customer subscription renewal using machine learning.

## Objectives

* Perform exploratory data analysis (EDA)
* Preprocess categorical and numerical features
* Predict subscription renewal using classification models
* Predict monthly spending using linear regression
* Compare model performance

## Machine Learning Models

* Decision Tree Classifier
* K-Nearest Neighbors (KNN)
* Linear Regression

## Libraries Used

* Pandas
* NumPy
* Matplotlib
* Scikit-learn
